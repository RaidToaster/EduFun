# EduFun Home Page

## Deskripsi
Halaman ini menyajikan gambaran singkat platform EduFun untuk jalur Data Science dan Network Security, memanfaatkan Bootstrap CDN tanpa penyesuaian build tambahan. Hero berisi ajakan bertindak, diikuti highlight kategori, artikel terbaru, ringkasan penulis, blok About Us, serta teaser Popular Page sebagai penghubung ke fitur lanjutan.

## Struktur Konten
1. Hero dan call-to-action menuju artikel dan Popular Page.
2. Kategori jalur NIM ganjil beserta deskripsi singkat.
3. Daftar artikel terbaru dengan tanggal, penulis, ringkasan, dan tombol read more.
4. Spotlight penulis untuk membangun kredibilitas.
5. Ringkasan About Us dengan tombol menuju informasi lanjutan.
6. Seksi Popular Page berisi tiga artikel populer dan indikator pagination dua halaman.
